Проект выполнен на Java в Intellij IDEA с программой для визуализации Gluon SceneBuilder.
Есть видео того, как работает данный проект

Настройки для проекта:
1) VM опции для Run/Debug configuration
--module-path ".\javafx-sdk-18.0.1\lib" --add-modules javafx.controls,javafx.fxml
2) Надо подключить библиотеку в настройках проекта (библиотека из .\javafx-sdk-18.0.1\lib)
